# CSS Animation Gallery

Interactive showcase of pure CSS animations.

## Animations Included
- Hover Lift (translateY)
- Rotate (rotate)
- Scale Pulse (scale)
- Fade In (opacity + translate)
- Slide In (translateX)
- Bounce (translateY)
- Spin (rotate)
- Glow Pulse (opacity + shadow)
- Flip X (scaleX)
- Shake (translateX)

## Performance Notes
All animations use `transform` and `opacity` properties only.  
These are GPU accelerated and avoid layout thrashing (reflows/repaints).
